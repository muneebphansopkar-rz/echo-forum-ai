# apps/

This directory is intentionally empty. Claude Code scaffolds your application
here during `/execute Phase 0 — Foundation` based on your `project/TECH-STACK.md`.

## After scaffolding, this will contain:

**Monorepo (backend + frontend):**
```
apps/
  api/    — your backend application
  web/    — your frontend application
```

**Single app (API-only or CLI):**
```
apps/
  api/    — your application
```

## Do not create files here manually.

Run `/plan` then `/execute` — Claude generates the correct scaffold for your stack.
